# JS-Sugar DateTime Library

JS-Sugar Date (JSS Date) is a immutable, multi-calendar, extensible, tree-shakable and lightweight date-time library for javascript.

## Main Features

- Immutable
- Multi-Calendar
- Widerange support
- Extensible
- Treeshakable
- Lightweight
